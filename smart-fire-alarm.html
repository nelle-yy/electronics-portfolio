Smart Fire Alarm & IoT Monitoring System

A wireless, IoT-based fire detection and monitoring system designed for smart buildings.

Project Highlights

  • Built a wireless star network topology with ESP32 and Raspberry Pi
  • Collected data from temperature, humidity, smoke, CO, and flame sensors
  • Implemented ANFIS-based machine learning for false alarm detection
  • Developed a web dashboard for real-time monitoring across rooms and buildings
  • Added push notifications and SMS alerts to building owners and fire departments
  • Tested successfully with local fire authorities

Technologies Used

Raspberry Pi • ESP32 • Python • C • ANFIS (Jupyter) • IoT Sensors

Outcome

Delivered a functional IoT fire alarm network that reduced false alerts and improved emergency response reliability.
