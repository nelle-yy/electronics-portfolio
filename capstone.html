Smart Window Automation System

A dual-module ESP32-based automated window control system. The system senses indoor and outdoor environmental conditions to automatically operate a window, improving comfort and air quality while ensuring weather safety.

NZ Diploma in Engineering (Level 6) — Capstone Project | In Progress (2026)

Project Highlights

  • Designed a two-module embedded system using dual ESP32 microcontrollers with ESP-NOW peer-to-peer wireless communication between indoor and outdoor nodes
  • Integrated DHT22 (temperature and humidity), SparkFun SEN-28869 anemometer (wind speed), and INA219 (current monitoring) sensors for multi-parameter environmental control
  • Implemented AUTO, MANUAL, and LOCK operating modes with interrupt-driven control logic and hardware override capability
  • Developed weather safety logic including wind speed threshold detection and storm lockout timer to force-close the window under unsafe conditions
  • Designed a BTS7960 H-bridge motor driver circuit to control a 12V DC linear actuator for window actuation with current feedback
  • Designed full custom PCB for both modules in Altium Designer, including schematic capture, component placement, and trace routing
  • Developed a web dashboard for real-time monitoring of sensor data and system status

System Architecture

Indoor Module: ESP32 master controller handling window actuation via BTS7960 motor driver, user interface (LCD display, LED indicators, buzzer, mode switch), INA219 current sensing, and ESP-NOW communication.

Outdoor Module: ESP32 remote node collecting outdoor temperature/humidity via DHT22, wind speed via anemometer (reed-switch, RJ11 interface), and transmitting data wirelessly to the indoor module via ESP-NOW.

Technologies Used

ESP32 • Altium Designer • Arduino IDE (C++) • ESP-NOW • DHT22 • SparkFun SEN-28869 Anemometer • INA219 • BTS7960 H-Bridge • 12V Linear Actuator • PCB Design • Web Dashboard

Status

Currently in progress. PCB layout under development in Altium Designer; firmware for manual mode, LCD, LED, and buzzer is functional. Full system integration and demonstration scheduled for Week 10.
