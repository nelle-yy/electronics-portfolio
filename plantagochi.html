Plantagochi – Interactive Plant Monitor

A fun, interactive plant monitor that gives visual feedback on soil moisture using an LED matrix.

Project Highlights

  • Measured soil moisture levels with analog sensors
  • Displayed plant “moods” (happy/sad faces) on an 8x8 LED matrix
  • Programmed logic for threshold detection and real-time updates

Technologies Used

Arduino • Soil Moisture Sensor • 8x8 LED Matrix • C++

Outcome

Created a fully functional prototype combining sensors, logic, and display for real-time interaction.
